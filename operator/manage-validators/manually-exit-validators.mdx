# Manually Exit Validators

Manual exits should only be used in specific scenarios: emergency, maintenance, or testing.

:::custom-warning[Validators Manager Required]
The wallet executing this command must be set as the [Validators Manager →](/operator/validators-manager) in the Vault settings. If the Validators Manager is set to the wallet address of the Relayer, you have to add the `--relayer-endpoint` parameter.
:::

```bash
./operator exit-validators
```

### `exit-validators` flags

| Flag | Description |
|------|-------------|
| <nobr>`--indexes`</nobr> | Comma-separated list of validator indexes to exit. Cannot be used with `--count` |
| <nobr>`--count`</nobr> | Number of validators to exit (default: all active). Cannot be used with `--indexes` |
| <nobr>`--relayer-endpoint`</nobr> | Relayer endpoint URL for API mode operations |
| <nobr>`--no-confirm`</nobr> | Skips confirmation prompts |
