# Vault Performance

StakeWise provides performance grades (Excellent, Good, Moderate, Bad) to help users evaluate Vault quality based on validator operational excellence. These grades reflect the average Vault performance over the last 7 days. Performance grades help identify well-operated Vaults but should be considered alongside other factors like fees, capacity, and operator reputation. [View detailed grade criteria →](/operator/vault-performance#performance-grades).

For operators seeking detailed information about performance calculations, scoring formulas, and optimization strategies, see [Vault Performance →](/operator/vault-performance).
