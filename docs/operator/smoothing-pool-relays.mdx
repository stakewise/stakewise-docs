# Smoothing Pool Relays

## Overview

If you selected the **Smoothing Pool** during Vault creation, you must use one or multiple StakeWise DAO-approved MEV relays. This ensures consistent high-quality contributions to the Smoothing Pool from every participating Vault.

:::custom-warning[Required Fee Recipient]
You must specify the Smoothing Pool address as the fee recipient for your Vault validators:

:::

:::custom-info[About MEV-Boost]
**MEV-Boost** is open-source middleware that enables validators to access a competitive block-building marketplace. It implements proposer-builder separation (PBS), allowing validators to receive blocks from multiple builders and select the most profitable option. This promotes competition, decentralization, and censorship-resistance in Ethereum's block production.
