# Run Operator Service V3

Before starting, make sure your node infrastructure and Vault meet the [prerequisites](./prerequisites). Then [install the Operator Service](./installation) binary or Docker image, [prepare the operator](./prepare-operator) by generating validator keys, creating a hot wallet, and uploading deposit data, and finally [start the service](./start-operator).
