# How to Create a Vault

:::custom-notes[Wallet Requirements]
You'll need an Ethereum wallet such as [Rabby Wallet ↗](https://rabby.io/).
The process will look very similar in other wallets.
:::

1. Navigate to the [StakeWise App ↗](https://app.stakewise.io) and go to the **Operate** page
2. Click the **Create Vault** button to begin the setup process
3. Choose the type of Vault you want to create:

- **Regular Vault** — runs validators and stakes assets. [Follow the Regular Vault guide →](./create-a-regular-vault)
- **Meta Vault** — does not run validators. Instead, it delegates assets to other Vaults (sub-vaults) that handle staking. [Follow the Meta Vault guide →](./create-a-meta-vault)
