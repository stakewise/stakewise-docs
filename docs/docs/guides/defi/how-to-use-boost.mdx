# How to Use Boost

:::custom-info[Prerequisites]
To use Boost, you need osETH in your wallet. osETH is StakeWise's liquid staking token — there are two ways to get it:
- [One-click staking →](/docs/guides/staking/one-click-staking): stake ETH on the [Stake page ↗](https://app.stakewise.io/) and receive osETH in one transaction
- [Staking with Vaults →](/docs/guides/staking/staking-with-vaults): stake ETH in a Vault, then mint osETH against your stake
:::

## Boosting Process

<div style={{ paddingLeft: '1.5rem' }}>

1. **Head to the Vault** page

<ThemedImage
  alt="boost_vault_page"
  sources={{
    light: require('./img/boost_vault_page.png').default,
    dark: require('./img/boost_vault_page_dark.png').default,
  }}
/>

2. **Check the Boost APY** displayed in the Vault to decide if it's a good time to enter
3. **Click plus (+) button** to start the boost process
4. **Enter the amount** of osETH to boost in the input field
5. **Review boost details** and confirm the transaction in your wallet. You will be asked to sign a spending approval for osETH when boosting for the first time.

</div>

:::custom-warning[Monitor Your Position]
Periodically check on your Vault position to ensure the Boost APY is positive. If you see a negative Boost APY for longer than 7 days in a row, consider unboosting your position. Stay informed for real-time updates via the [StakeWise Discord ↗](https://discord.com/invite/2BSdr2g).
:::

## Unboosting Process

<div style={{ paddingLeft: '1.5rem' }}>

1. **Head to the Vault** page
2. **Click minus (–) button** to start the unboost process
3. **Enter the amount** of osETH to unboost in the input field
4. **Confirm the transaction** – Your request enters the unboost queue, which takes up to ~10 days to complete. Only one unboost request can be active at a time.
5. **Claim your assets** – Once processing is complete, click the **Claim** button on the Vault page. Claim as soon as assets become available to maximize your rewards.

</div>
